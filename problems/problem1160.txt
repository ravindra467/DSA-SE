b'Problem\n\xc2\xa0\nAs we all know, a substring is a contiguous subsequence of characters of a word. For example, the substrings of the word cat would be c,a,t,ca,at,cat.  \nGiven a String  S composed of only lower case characters, and a substring  X  also of lower case characters, your job is to remove all occurrences of X from S .    For example, if the String is \xe2\x80\x9cbanana\xe2\x80\x9d and the substring is an, the resulting String would be \xe2\x80\x9cba\xe2\x80\x9d.                                                                                                       If the resulting String has zero length, print \xe2\x80\x9c0\xe2\x80\x9d (without quotes).\n\n\xc2\xa0\nInput\nThe first Line of Input will contain an Integer T, Denoting the number of Test Cases. The next 2T  Lines will contain two Strings  S and X, alternatively.\nOutput\nFor each test case, print the resulting String on a new line.\nConstraints\nShould contain all the constraints on the input data that you may have. Format it like:\n\n1 \xe2\x89\xa4 |S| \xe2\x89\xa4 15 (30 Points)\n1 \xe2\x89\xa4 |S| \xe2\x89\xa4 1000 (70 Points)\n1 \xe2\x89\xa4 |X| \xe2\x89\xa4 |S| \n\n\xc2\xa0\nExample\nInput:\n2bananaanlolwutlolwutlolwutlol\nOutput:\nbawutwutwut\n\n\xc2\xa0\nExplanation\nRemoving all occurrences of \xe2\x80\x98an\xe2\x80\x99 from \xe2\x80\x98banana\xe2\x80\x99 results in \xe2\x80\x98ba\xe2\x80\x99.     Similarly, removing all occurrences of \xe2\x80\x98lol\xe2\x80\x99 from \xe2\x80\x98lolwutlolwutlolwut\xe2\x80\x99 results in \xe2\x80\x98wutwutwut\xe2\x80\x99.\n \nThe Input and Output Format are to be followed STRICTLY. It is advised to use BufferedReader for this problem.\n'
